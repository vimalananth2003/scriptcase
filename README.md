# Scriptcase Landing Page

## 📌 Project Overview
This project is a responsive landing page designed using HTML and CSS. It contains a banner image, event information, a registration button, and a footer with social media icons.

## 🚀 Features
- Header with Scriptcase logo
- Banner image
- Event description section
- Event details (Date, Time, Speaker)
- Registration button
- Social media icons using Font Awesome
- Footer with copyright information

## 🛠️ Technologies Used
- HTML
- CSS
- Font Awesome
